Envvia Admin Portal

This is the internal admin interface that manages the content for the online store. While it does not need to be as subdued as the store, we maintained the same light colors for consistency. And also because the admin will be spending a lot of time here, eye fatigue was something we considered. A dark mode is in the works.

This panel has a very utilitarian feel. It uses our internal design system called Facet UI. There is no mobile view as the stakeholders said that they would rarely ever manage content in that form factor.
