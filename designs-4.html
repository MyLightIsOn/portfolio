Facet UI Design System

This design system and component library was developed to streamline our client projects. After repeatedly building the same components from scratch, we decided to create a unified system.

We named it "Facet" to symbolize the diverse styles and brands we work with. It features a minimal color palette since we typically use client-specific colors. The system is intentionally designed to be highly adaptable to align seamlessly with our clients' branding.
